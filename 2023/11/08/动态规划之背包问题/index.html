😂动态规划之背包问题

经典的背包问题，期末考试复习用

一、01背包问题

Description

给定 N 件物品和一个容量为 V 的背包。每件物品只能使用一次。

第i件物品的体积是vi，价值是wi。

你的任务是决定将哪些物品装入背包，以使得这些物品的总体积不超过背包容量，同时总价值最大。请输出最大价值。

Input

  • 第一行包含两个整数，分别为N和V，表示物品数量和背包容量。
  • 接下来的 N 行，每行包含两个整数vi和wi，表示第i件物品的体积和价值。

Output

输出一个整数，表示最大价值。

Sample Input 1

4 5
1 2
2 4
3 4
4 5

Sample Output 1

8

01背包在于每件物品只有一个，要么装，要么不装

1、思路
每个物品有两个状态，装与不装。构造dp数组记录每一种情况

根据以上例子构建dp数组，第一维代表第i个物品，第二维代表背包当前容积，所以**dp[i][j]代表i个物品，背包容积为j时能够装到的最大价值**

下面是模拟过程的矩阵

i j012345
0000000
1022222
2024666
3024668
4024668

状态转移方程为

if 当前物品体积 > 当前背包容积：

dp[i][j] = dp[i-1][j]

else

dp[i][j] = max(dp[i-1][j],dp[i-1][j-当前物品体积]+当前物品价值)   //判断装这个物品是否会价值变大
2、代码
#include<iostream>
#include<vector>
using namespace std;

int main(){
    int N,V;
    vector<int> v(N+1);
    vector<int> w(N+1);
    cin >> N >> V;
    for(int i = 1;i <= N;i++){
        cin >> v[i] >> w[i];
    }
    vector<vector<int>> dp(N+1,vector<int>(V+1));
    for(int i = 1;i <= N;i++){
        for(int j = 1;j <= V;j++)
        {  
            if(j < v[i])       //如果前物品体积大于当前背包容积，则取未装该物品时能够得到的最大价值
            dp[i][j] = dp[i-1][j];
            else             //否则判断是否值得装这件物品
            dp[i][j] = max(dp[i-1][j],dp[i-1][j-v[i]]+w[i]);

        }
      
    }
    cout << dp[N][V] << endl;
    return 0;

}

二、完全背包

Description

有一个背包，它的容量为V。还有N种不同类型的物品，每种物品都有无限数量可供选择。

每种物品的体积为v_i，价值为w_i。

请你确定如何选择物品放入背包，以确保物品的总体积不超过背包的容量，并且物品的总价值达到最大。你需要输出最大的价值。

Input

  • 第一行：两个整数，分别代表物品的种数N和背包的容积V。
  • 接下来的N行：每行包含两个整数，分别代表一种物品的体积v_i和价值w_i。

Output

  • 输出一个整数，表示能够获得的最大价值。

Sample Input 1

4 5
1 2
2 4
3 4
4 5

Sample Output 1

10
1、思路

物品还是只有两种状态，装与不装，不过现在每件物品有无限多个而已

此时**dp[i][j]代表装第i种物品，背包容积为j时能够装到的最大价值**

模拟矩阵

i j012345
0000000
10246810
20246810
30246810
40246610

状态转移方程为

for(int i = 1;i <= N;i++){
        for(int j = 1;j <= V;j++){
            if(v[i] > j)
            dp[i][j] = dp[i-1][j];
            else{
                for(int k = 0;k * v[i] <= j; k++)
                 dp[i][j] = max(dp[i-1][j],dp[i-1][j-k*v[i]]+k*w[i]);
            }
        }
    }

二、多重背包

Description

你有一个背包，容量为V，以及N种不同类型的物品。每种物品有限数量供选择。

对于第i种物品：

  • 它有s_i件可用。
  • 每件的体积是v_i。
  • 每件的价值是w_i。

请确定如何选择物品放入背包，以确保所选物品的总体积不超过背包的容量，并使所选物品的总价值最大化。输出能够获得的最大价值。

Input

  • 第一行：两个整数，分别代表物品的种数N和背包的容积V。
  • 接下来的N行：每行包含三个整数，代表一种物品的体积v_i、价值w_i和数量s_i。

Output

  • 输出一个整数，表示能够获得的最大价值。

Sample Input 1

4 5
1 2 3
2 4 1
3 4 3
4 5 2

Sample Output 1

10
1、思路

物品还是只有两种状态，装与不装，不过现在每件物品有多个而已

此时**dp[i][j]代表装第i种物品，背包容积为j时能够装到的最大价值**

状态矩阵如下：

i j012345
0000000
1024666
20246810
30246610
40246610

转移方程略有变化

if(当前物品体积 > 当前背包容积)
	dp[i][j] = dp[i-1][j];
else{
                当前背包可以装下物品i的数量 = j/v[i];
                if(当前背包可以装下物品i的数量 <= 物品i的数量)
                dp[i][j] = max(dp[i-1][j],dp[i][j- 当前背包可以装下物品i的数量 * v[i] ]+当前背包可以装下物品i的数量 * w[i]);
                else   
                dp[i][j] = max(dp[i-1][j],dp[i][j-物品i的数量*v[i]] + 物品i的数量 * w[i]);
}
2、代码
#include<iostream>
#include<vector>
using namespace std;


int main(){
    int N,V;
    cin >> N >> V;
    vector<int> v(N+1);
    vector<int> w(N+1);
    vector<int> t(N+1);
    
    for(int i = 1;i <= N;i++){
        cin >> v[i] >> w[i] >> t[i];
    }
    vector<vector<int>> dp(N+1,vector<int>(V+1));
    for(int i = 1;i <= N;i++){
        for(int j = 1;j <= V;j++){
            if(v[i] > j)
            dp[i][j] = dp[i-1][j];
            else{
                for(int k = 0;k <= t[i] && k * v[i] <= j; k++)
                 dp[i][j] = max(dp[i][j],dp[i-1][j-k*v[i]]+k*w[i]);
            }
        }
    }
    cout << dp[N][V] << endl;
    return 0;
}
