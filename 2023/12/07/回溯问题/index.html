回溯法，一般可以解决如下几种问题：

  • 组合问题：N个数里面按一定规则找出k个数的集合
  • 切割问题：一个字符串按一定规则有几种切割方式
  • 子集问题：一个N个数的集合里有多少符合条件的子集
  • 排列问题：N个数按一定规则全排列，有几种排列方式
  • 棋盘问题：N皇后，解数独等等

回溯法解决的问题都可以抽象为树形结构，因为回溯法解决的都是在集合中递归查找子集，集合的大小就构成了树的宽度，递归的深度，都构成的树的深度。

回溯算法模版

void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}

刷题：

77. 组合

给定两个整数 n 和 k，返回范围 [1, n] 中所有可能的 k 个数的组合。

你可以按 任何顺序 返回答案。

示例 1：

输入：n = 4, k = 2
输出：
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]

示例 2：

输入：n = 1, k = 1
输出：[[1]]

提示：

  • 1 <= n <= 20
  • 1 <= k <= n

思路

组合问题一般用回溯解，n个数里面挑选k个数组合，抽象为树，高度为两层。第一层是从1,2,3,4里面取一个数，第二层是从父节点之后的数中取，避免重复，例如取2点的子节点智能从3,4中选一个数。这个可以用一个参数来限定。我们严格按照模版来实现。

代码

#include<iostream>
#include<vector>
using namespace std;
vector<vector<int>> res;
vector<int> path;
void backtracking(int n,int k,int index){
    //终止条件
    if(path.size() == k){
        res.push_back(path);  //存放结果
        return;
    }
    for(int i = index;i <= n;i++){    //每层集合中元素取决于上一层所选元素的后面的元素
        path.push_back(i);          //处理节点
        backtracking(n,k,i+1);     //递归 
        path.pop_back();            //回溯
    }
}
int main(){
    int n,k;
    cin >> n >> k;
    backtracking(n,k,1);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < k;j++){
            cout << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

很好地解决了上述问题，可是回溯和递归是离不开的，回溯实际上时枚举所有解，所以学会如何剪枝是优化算法的关键，上述代码完全没有考虑剪枝的事情，可以考虑一下，4个数里找两个数的组合，所以第一轮挑选时最大从3开始，还有4可以选，所以第一轮4这一支可以剪除。这个例子中优化效果还不是很明显。想一下，如果是4个数里选4个数的组合，那么第一轮就可以排除2,3,4三个节点，只能选1，第二轮排除3,4两个节点，选2，第三轮排除4节点，只能选3，最后一轮选4，遇到这样的问题时，剪枝对算法的优化程度是很大的。

所以可以在backtracking函数中对i的取值范围进行优化，根据上面分析，i<=n可以改为i <= n - (k-path.size()) + 1，为什么有个+1呢，因为包括起始位置，我们要是一个左闭的集合。

优化

后代码如下

#include<iostream>
#include<vector>
using namespace std;
vector<vector<int>> res;
vector<int> path;
void backtracking(int n,int k,int index){
    //终止条件
    if(path.size() == k){
        res.push_back(path);  //存放结果
        return;
    }
    for(int i = index;i <= n - (k - path.size()) + 1;i++){    //每层集合中元素取决于上一层所选元素的后面的元素
        path.push_back(i);          //处理节点
        backtracking(n,k,i+1);     //递归 
        path.pop_back();            //回溯
    }
}
int main(){
    int n,k;
    cin >> n >> k;
    backtracking(n,k,1);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < k;j++){
            cout << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

216. 组合总和 III

找出所有相加之和为 n 的 k 个数的组合，且满足下列条件：

  • 只使用数字1到9
  • 每个数字 最多使用一次

返回 所有可能的有效组合的列表 。该列表不能包含相同的组合两次，组合可以以任何顺序返回。

示例 1:

输入: k = 3, n = 7
输出: [[1,2,4]]
解释:
1 + 2 + 4 = 7
没有其他符合的组合了。

示例 2:

输入: k = 3, n = 9
输出: [[1,2,6], [1,3,5], [2,3,4]]
解释:
1 + 2 + 6 = 9
1 + 3 + 5 = 9
2 + 3 + 4 = 9
没有其他符合的组合了。

示例 3:

输入: k = 4, n = 1
输出: []
解释: 不存在有效的组合。
在[1,9]范围内使用4个不同的数字，我们可以得到的最小和是1+2+3+4 = 10，因为10 > 1，没有有效的组合

提示:

  • 2 <= k <= 9
  • 1 <= n <= 60

思路

这种组合问题还是回溯啊，只不过得多加个参数，判断一下路径和是否为条件值。

代码

#include<iostream>
#include<vector>
using namespace std;
vector<vector<int>> res;
vector<int> path;
void backtracking(int targetSum,int k,int index){
    //剪枝
    if(targetSum < 0)
    return;
    //终止条件
    if(path.size() == k){
        if(targetSum == 0)
        res.push_back(path);  //存放结果
        return;
    }
    for(int i = index;i <= 9 - (k - path.size()) + 1;i++){    //每层集合中元素取决于上一层所选元素的后面的元素
        path.push_back(i);          //处理节点
        targetSum -= i;
        backtracking(targetSum,k,i+1);     //递归 
        path.pop_back();            //回溯
        targetSum += i;
    }
}
int main(){
    int n,k;
    cin >> n >> k;
    backtracking(n,k,1);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < k;j++){
            cout << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

顺便参照上题剪了枝，同时还有一处剪枝操作，当targetSum值为负数的时候，再往后已经没有意义，直接返回🥳

17.电话号码的字母组合

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。答案可以按 任意顺序 返回。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

示例 1：

输入：digits = "23"
输出：["ad","ae","af","bd","be","bf","cd","ce","cf"]

示例 2：

输入：digits = ""
输出：[]

示例 3：

输入：digits = "2"
输出：["a","b","c"]

提示：

  • 0 <= digits.length <= 4
  • digits[i] 是范围 ['2', '9'] 的一个数字。

思路

这个和上面组合问题稍有不同，上面组合问题的结果都是在一个集合里组合，这个是多个集合组合。首先得做好数字到字母组合的映射，然后每次从一个组合挑选一个字母然后组合为最后的字符串，如何控制所选择字母的位次，做到不重不漏比较重要

确定回溯函数参数

首先题目中给出digits字符串，还要有一个int型的index，记录遍历第几个字符，也可以表示树的深度。

vector<string> res;
string s;
void backtracking(string digits,int index);
确定终止条件

很显然，以“23”为例，树有两层，所以路径长度和输入字符串长度一样时，即可终止

if(path.size() == digits.size()){
    res.push_back(path);
    return;
}
确定单层循环逻辑

首先用index获取当前字符串，然后依次遍历字符串中字符，每遍历一个字符，递归进入下一次，即从下一个字符串中挑选一个字符进行组合。

string s = tel[digits[index]-'0'];
for(int i = 0;i < s.size();i++){
    path.push_back(s[i]);
    backtracking(digits,index+1);
    path.pop_back();
}

39. 组合总和

给你一个 无重复元素 的整数数组 candidates 和一个目标整数 target ，找出 candidates 中可以使数字和为目标数 target 的 所有 不同组合 ，并以列表形式返回。你可以按 任意顺序 返回这些组合。

candidates 中的 同一个 数字可以 无限制重复被选取 。如果至少一个数字的被选数量不同，则两种组合是不同的。

对于给定的输入，保证和为 target 的不同组合数少于 150 个。

示例 1：

输入：candidates = [2,3,6,7], target = 7
输出：[[2,2,3],[7]]
解释：
2 和 3 可以形成一组候选，2 + 2 + 3 = 7 。注意 2 可以使用多次。
7 也是一个候选， 7 = 7 。
仅有这两种组合。

示例 2：

输入: candidates = [2,3,5], target = 8
输出: [[2,2,2,2],[2,3,3],[3,5]]

示例 3：

输入: candidates = [2], target = 1
输出: []

提示：

  • 1 <= candidates.length <= 30
  • 2 <= candidates[i] <= 40
  • candidates 的所有元素 互不相同
  • 1 <= target <= 40

思路

这一道题和之前的组合总和有不同，首先每个数字可以循环使用，其次还要去重。还是在脑海中用树模拟，

确定递归参数

首先candidates数组必须，target必须，需要用target==0判断是否符合条件，以及可以用target判断终止条件。startIndex控制遍历范围。

vector<vector<int>> res;
vector<int> path;
void backtracking(vector<int> candidates,int target,int startindex);
确定终止条件

target值，每次要减去加入路径中的数，当target < 0 时可以终止，当target==0时，先将路径加入res中再终止

if(target < 0)
    return;
if(target == 0)
{
    res.push_back(path);
    return;
}
确定单层搜索逻辑

搜索范围由上一层选取数据决定，从上层选取数据开始往后遍历而不是从上层选取数据下一个遍历，因为每个数字可以循环选取，至于为什么不每次都从头选取，当然是为了避免重复。

for(int i = startindex;i < candidates.size();i++){
    path.push_back(candidates[i]);
    target -= candidates[i];
    backtracking(candidates,target,i);
    path.pop_back();
    target += candidates[i];
}

40. 组合总和 II

给定一个候选人编号的集合 candidates 和一个目标数 target ，找出 candidates 中所有可以使数字和为 target 的组合。candidates 中的每个数字在每个组合中只能使用 一次 。

注意：解集不能包含重复的组合。

示例 1:

输入: candidates = [10,1,2,7,6,1,5], target = 8,
输出:
[
[1,1,6],
[1,2,5],
[1,7],
[2,6]
]

示例 2:

输入: candidates = [2,5,2,1,2], target = 5,
输出:
[
[1,2,2],
[5]
]

提示:

  • 1 <= candidates.length <= 100
  • 1 <= candidates[i] <= 50
  • 1 <= target <= 30

思路

这个和上一题的区别在于多了去重的要求，关于去重，可以分为树枝去重和树层去重，根据本题要求，树枝去重不必要，重要的是树层去重，首先对candidates数组排序，需要有一个used数组记录candidates数组的使用情况，用来判断上一个数字是父节点还是兄弟节点。如下图：来自代码随想录

代码

#include <iostream>
#include <vector>
#include<algorithm>
using namespace std;
vector<vector<int>> res;
vector<int> path;
int used[110];   //用于记录数组元素使用情况，判断当前处理的节点是上一个的子节点还是兄弟节点

void backtracking(vector<int> candidates,int target,int startIndex){
    if(target < 0)
    return;
    if(target == 0)
    {
        res.push_back(path);
        return;
    }
    for(int i = startIndex;i < candidates.size();i++){
        if(i > 0 && candidates[i] == candidates[i-1] && used[i-1] == 0) //如果当前值和上一个节点值相同，并且还是上一个节点的兄弟节点的话，就不用进入
        continue;
        used[i] = 1;
        path.push_back(candidates[i]);
        backtracking(candidates,target - candidates[i],i + 1);
        path.pop_back();
        used[i] = 0;
    }
}
int main()
{
    int n,target;
    cin >> n >> target;
    vector<int> candidates(n);
    for(int i = 0;i < n;i++){
        cin >> candidates[i];
    }
    sort(candidates.begin(),candidates.end());
    backtracking(candidates,target,0);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < res[i].size();j++){
            cout << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

131. 分割回文串

给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是 回文串 。返回 s 所有可能的分割方案。

回文串 是正着读和反着读都一样的字符串。

示例 1：

输入：s = "aab"
输出：[["a","a","b"],["aa","b"]]

示例 2：

输入：s = "a"
输出：[["a"]]

提示：

  • 1 <= s.length <= 16
  • s 仅由小写英文字母组成

思路

确定递归参数

说先一个vector<vector> res来存放最终结果，vectorpath存放路径结果，int startIndex用来控制切割的点。

vector<vector<string>> res;
vector<string>path;
void backtracking(string s,int startIndex);
确定递归终止条件

当切割点到了字符串最后说明找到了切割方法，此时是本层递归的终点。

if(startIndex == s.size()){
    res.push_back(path);
    return;
}
单层搜索逻辑

在for(int i = startIndex;i < s.size();i++)中[startIndex,i]就是要切割的字串，判断该字串是否为回文串，是就加入path，递归，不是，循环进入下一个（i++）

for(int i = startIndex;i < s.size();i++){
    if(!IsPalindrome(s, startIndex, i)){
        continue;
    }
    string str = s.substr(startIndex, i - startIndex + 1);
    path.push_back(str);
    backtracking(s,i+1);
    path.pop_back();
}
判断是否为回文串
bool IsPalindrome(string s,int start,int end){
    while(start < end){
        if(s[start] == s[end]){
            start++;
            end--;
        }else
            return false;
    }
    return true;
}

代码

#include<bits/stdc++.h>
using namespace std;
vector<vector<string>> res;
vector<string> path;
bool IsPalindrome(string s,int start,int end){
    while(start < end){
        if(s[start] == s[end])
        {
            start++;
            end--;
        }
        else
        return false;
    }
    return true;
}
void backtracking(string s,int startIndex){
    if(startIndex == s.size())
    {
        res.push_back(path);
        return;
    }
    for(int i = startIndex;i < s.size();i++){
        if(!IsPalindrome(s,startIndex,i))
          continue;
        string str = s.substr(startIndex,i - startIndex + 1);
        path.push_back(str);
        backtracking(s,i + 1);
        path.pop_back();
    }
}
int main(){
    string s;
    cin >> s;
    backtracking(s,0);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < res[i].size();j++){
            cout  << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

优化

对于判断回文串还是有可以优化的地方的，具体来说, 给定一个字符串s, 长度为n, 它成为回文字串的充分必要条件是s[0] == s[n-1]且s[1:n-1]是回文字串。可以用动态规划的方法通过查询字串是否为回文串来判断。

#include<bits/stdc++.h>
using namespace std;
vector<vector<string>> res;
vector<string> path;
vector<vector<bool>> isPalindrome; // 放事先计算好的是否回文子串的结果
void computeIsPalindrome(string s){
   isPalindrome.resize(s.size(), vector<bool>(s.size(), false)); // 根据字符串s, 刷新布尔矩阵的大小
   for (int i = s.size() - 1; i >= 0; i--) { 
            // 需要倒序计算, 保证在i行时, i+1行已经计算好了
            for (int j = i; j < s.size(); j++) {
                if (j == i) {isPalindrome[i][j] = true;}
                else if (j - i == 1) {isPalindrome[i][j] = (s[i] == s[j]);}
                else {isPalindrome[i][j] = (s[i] == s[j] && isPalindrome[i+1][j-1]);}
            }
        }
}
void backtracking(string s,int startIndex){
    if(startIndex == s.size())
    {
        res.push_back(path);
        return;
    }
    for(int i = startIndex;i < s.size();i++){
        if(!isPalindrome[startIndex][i])
          continue;
        string str = s.substr(startIndex,i - startIndex + 1);
        path.push_back(str);
        backtracking(s,i + 1);
        path.pop_back();
    }
}
int main(){
    string s;
    cin >> s;
    computeIsPalindrome(s);
    backtracking(s,0);
    for(int i = 0;i < res.size();i++){
        for(int j = 0;j < res[i].size();j++){
            cout  << res[i][j] << " ";
        }
        cout << endl;
    }
    return 0;
}

93. 复原 IP 地址

有效 IP 地址 正好由四个整数（每个整数位于 0 到 255 之间组成，且不能含有前导 0），整数之间用 '.' 分隔。

  • 例如："0.1.2.201" 和 "192.168.1.1" 是 有效 IP 地址，但是 "0.011.255.245"、"192.168.1.312" 和 "192.168@1.1" 是 无效 IP 地址。

给定一个只包含数字的字符串 s ，用以表示一个 IP 地址，返回所有可能的有效 IP 地址，这些地址可以通过在 s 中插入 '.' 来形成。你 不能 重新排序或删除 s 中的任何数字。你可以按 任何 顺序返回答案。

示例 1：

输入：s = "25525511135"
输出：["255.255.11.135","255.255.111.35"]

示例 2：

输入：s = "0000"
输出：["0.0.0.0"]

示例 3：

输入：s = "101023"
输出：["1.0.10.23","1.0.102.3","10.1.0.23","10.10.2.3","101.0.2.3"

提示：

  • 1 <= s.length <= 20
  • s 仅由数字组成

思路

本题和切割回文串相似，需要写一个判断当前所切字符串是否符合ip地址要求的函数，其他的大差不差。

确定递归参数

需要一个vector<string> res接收符合条件的ip。

参数首先字符串s传进去，和上一题一样需要一个起始下标startIndex，最后需要有一个pointNum记录插入的点数。

vector<string> res;
void backtracking(string s,int startIndex,int pointNum);
确定终结条件

当pointNum==3时说明当前字符串已经分成4段，这时判断最后一段是否符合Ip要求，符合就把它送入res中

if(pointNum == 3){
    if(IsIp(s,startIndex,s.size()-1))
    {
        res.push_back(s);
        return;
    }
    return;
}
单层搜索过程

对于每层首先判断当前层字符串是否符合要求，符合在进行下一步，不然跳出循环

for(int i = startIndex;i < s.size();i++){
    if(IsIp(s,startIndex,i)){
        s.insert(s.begin()+i+1,'.');
        backtracking(s,i+2,pointNum+1); //为啥加2,因为s插入了'.'，所以下一步切割点需要往后挪2个
        s.erase(s.begin()+i+1);
    }else
        break;
}
判断函数
bool IsIp(string s,int start,int end){
    if(start > end)
        return false;
    if(s[start] == '0' && end != start)
        return false;
    int sum = 0;
    for(int i = 0;i <= end;i++){
        if(s[i] >= '0' && s[i] <= '9')
        {
            sum = sum * 10 + (s[i] - '0');
            if(sum > 255)
                return false;
        }else
            return false;
    }
    return true;
}

代码

#include <bits/stdc++.h>
using namespace std;
vector<string> res;
// 判断给定字符串s中从索引start到end之间的子串是否是一个合法的IP地址
bool IsIp(string s, int start, int end)
{
    // 如果start大于end，则不是一个合法的IP地址
    if(start > end)
        return false;

    // 如果子串的起始字符是'0'且子串长度大于1，则不是一个合法的IP地址
    if(s[start] == '0' && start != end)
        return false;

    int sum = 0; // 用于计算IP地址每个段的数值

    // 遍历子串，计算每个段的数值
    for(int i = start; i <= end; i++)
    {
        // 如果子串中的字符不是数字，则不是一个合法的IP地址
        if(s[i] < '0' || s[i] > '9')
            return false;

        sum = sum * 10 + s[i] - '0'; // 将字符串转换成数字
        // 如果某个段的数值大于255，则不是一个合法的IP地址
        if(sum > 255)
            return false;
    }

    // 返回true表示是一个合法的IP地址
    return true;
}
    void backtracking(string s, int startIndex, int pointNum)
    {
        // 递归回溯函数，用于生成IP地址
        // 参数s为输入的字符串
        // 参数startIndex为起始索引
        // 参数pointNum为已点的数

        if (pointNum == 3)
        {
            // 如果点的数量已经达到了3个
            if (IsIp(s, startIndex, s.size() - 1))
            {
                // 如果当前字符串是合法的IP地址
                res.push_back(s);
                return; // 将其加入结果集合中并返回
            }
            return; // 不满足条件，返回
        }
        
        // 遍历字符串，从起始索引到末尾
        for (int i = startIndex; i < s.size(); i++)
        {
            // 如果当前子字符串是合法的IP地址
            if (IsIp(s, startIndex, i))
            {
                // 在第i+1位插入点，并继续递归调用回溯函数
                s.insert(s.begin() + i + 1, '.');
                backtracking(s, i + 2, pointNum + 1);
                s.erase(s.begin() + i + 1); // 回溯，移除插入的点
            }
        }
    }
    int main()
    {
        string s;
        cin >> s;
        backtracking(s, 0, 0);
        for (int i = 0; i < res.size(); i++)
        {
            cout << res[i] << endl;
        }
            res.clear();
            return 0;       
    }
